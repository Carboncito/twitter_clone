# Twitter Clone Mobile App

## Installation

Run the next command to install modules:

```bash
yarn install
```

## Full clean pods

In ios folder run the next commands:
```bash
rm -rf ~/Library/Caches/CocoaPods
rm -rf Pods
rm -rf ~/Library/Developer/Xcode/DerivedData/*
pod deintegrate
pod setup
pod install
```
